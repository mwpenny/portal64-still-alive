# damage_player

Damages the player.

## Syntax

```
damage_player AMOUNT [COLOR]
```

## Arguments

| Name               | Description                                                                            |
| ------------------ | -------------------------------------------------------------------------------------- |
| `AMOUNT`           | The amount of damage to apply to the player                                            |
| `COLOR` (optional) | The color to flash on screen when the damage is applied. Defaults to `FFFFFF` (white). |

## Notes

The player's maximum health is 100.
